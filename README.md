# Fotche Updates

This repository hosts the Sparkle appcast and release artifacts for
[Fotche](https://fotche.app), the macOS font manager.

## What's here

- `appcast.xml` - the update feed read by Sparkle inside Fotche.
- GitHub Releases - signed `.zip` archives of each Fotche build.

## What's NOT here

Fotche's source code lives in a separate private repository. This repo only
carries the metadata Sparkle needs to deliver updates.

## Getting Fotche

Download the app from [fotche.app](https://fotche.app).
